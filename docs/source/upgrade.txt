Upgrade Anaconda packages.

**usage**: ``conda upgrade [-h] [--confirm {yes,no}] [--dry-run] [-n NAME | -p PREFIX] [package_name [package_name ...]]``

*package_name*
	names of packages to upgrade (default: all packages)

optional arguments:
  -h, --help            show this help message and exit
  --confirm     ask for confirmation before upgrading packages (default: yes)
  --dry-run             display packages to be modified, without actually exectuting
  -n NAME, --name NAME  name of new directory (in ROOT_DIR/envs) to upgrade packages in
  -p PREFIX, --prefix PREFIX
                        full path to Anaconda environment to upgrade packages in (default: ROOT_DIR)
