Activate available packages in the specified Anaconda enviropnment.

**usage**: ``conda activate [-h] [--confirm {yes,no}] [--dry-run] [-p PREFIX] canonical_name [canonical_name ...]``

*canonical_name*
	canonical name of package to activate in the specified Anaconda environment

optional arguments:
  -h, --help            show this help message and exit
  --confirm     ask for confirmation before activating packages in Anaconda environment (default: yes)
  --dry-run             display packages to be modified, without actually executing
  -p PREFIX, --prefix PREFIX
                        Anaconda environment to activate packages ini (default: ROOT_DIR)
